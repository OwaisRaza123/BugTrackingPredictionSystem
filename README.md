# Bug Tracking and Prediction System

## Overview

Bug Tracking and Prediction System is a web-based application developed using Flask, MySQL, and Machine Learning. The system helps testers report bugs, developers resolve them, and predicts bug severity and estimated resolution time using machine learning.

## Features

- User Authentication
- Role-Based Access (Admin, Developer, Tester)
- Project Management
- Bug Reporting
- Bug Assignment
- Bug Tracking
- Machine Learning Severity Prediction
- Resolution Time Prediction
- Bug History
- Dashboard & Reports

## Technologies Used

- Python
- Flask
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
- Scikit-learn
- Pandas
- NumPy

## Machine Learning

- TF-IDF Vectorizer
- Naive Bayes (Severity Prediction)
- Random Forest Classifier (Resolution Prediction)

## Installation

```bash
git clone <repository-url>
cd BugTrackingPredictionSystem

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python app.py
```
